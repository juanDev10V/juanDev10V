Cloud Computing

¿Qué es Cloud Computing?

Es el suministro de servicios de computación, como almacenamiento y procesamiento, a través de Internet bajo demanda.

¿Qué es SaaS, PaaS, e IaaS?

Son modelos de servicio en la nube que ofrecen software, plataformas y recursos de infraestructura, respectivamente.

¿Qué es Virtualización?

Es la tecnología que permite crear versiones virtuales de recursos físicos, como servidores y almacenamiento.

¿Qué es Docker?

Es una plataforma que utiliza contenedores para desarrollar, distribuir y ejecutar aplicaciones de manera eficiente.

AnyDesk

AnyDesk es una plataforma ligera y rápida para acceso remoto entre PCs, ideal para soporte técnico y colaboración a distancia.
